# Funding Opportunities API

Base prefix: `/funding-opportunities`

All endpoints require `funding-opportunities` whitelist.

| Method | Path | Description | File |
|--------|------|-------------|------|
| GET | /funding-opportunities | List funding opportunities | [get_funding-opportunities.md](get_funding-opportunities.md) |
| GET | /funding-opportunities/count | Get count | [get_funding-opportunities_count.md](get_funding-opportunities_count.md) |
| GET | /funding-opportunities/{opportunity_id} | Get by ID | [get_funding-opportunities_{opportunity_id}.md](get_funding-opportunities_{opportunity_id}.md) |
| POST | /funding-opportunities | Create opportunity | [post_funding-opportunities.md](post_funding-opportunities.md) |
| PUT | /funding-opportunities/{opportunity_id} | Update opportunity | [put_funding-opportunities_{opportunity_id}.md](put_funding-opportunities_{opportunity_id}.md) |
| DELETE | /funding-opportunities/{opportunity_id} | Delete opportunity | [delete_funding-opportunities_{opportunity_id}.md](delete_funding-opportunities_{opportunity_id}.md) |
