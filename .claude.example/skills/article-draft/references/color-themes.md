# Color Themes

All slides use a **white background** (`bg-white`) with `text-gray-900` body text. Each theme provides a distinct **accent color palette** for titles, cards, and icons. Pick the theme that best fits the topic.

All themes use subtitle `text-gray-500` and footer `text-gray-400`.

## Theme 1: Midnight (tech, engineering, systems)
- Title accent: `text-blue-600`
- Cards: `bg-blue-50 border border-blue-200`
- Accent palette: blue, emerald, amber, purple, cyan

## Theme 2: Ocean (data, analytics, AI, intelligence)
- Title accent: `text-cyan-600`
- Cards: `bg-sky-50 border border-sky-200`
- Accent palette: cyan, teal, sky, blue, indigo

## Theme 3: Forest (growth, sustainability, HR, green topics)
- Title accent: `text-emerald-600`
- Cards: `bg-emerald-50 border border-emerald-200`
- Accent palette: emerald, teal, lime, green, cyan

## Theme 4: Sunset (business, creative, marketing, design)
- Title accent: `text-amber-600`
- Cards: `bg-orange-50 border border-orange-200`
- Accent palette: amber, orange, rose, pink, yellow

## Theme 5: Violet (AI, innovation, future, research)
- Title accent: `text-purple-600`
- Cards: `bg-purple-50 border border-purple-200`
- Accent palette: purple, violet, fuchsia, pink, indigo

## Theme 6: Slate (corporate, professional, legal, formal)
- Title accent: `text-slate-700`
- Cards: `bg-gray-50 border border-gray-200 shadow-sm`
- Accent palette: blue, slate, indigo, gray, emerald

## Theme 7: Warm (supply chain, operations, inventory, logistics)
- Title accent: `text-yellow-600`
- Cards: `bg-amber-50 border border-amber-200`
- Accent palette: yellow, amber, orange, stone, warm-gray

**IMPORTANT**: Vary the theme across different invocations. Do NOT always default to Theme 1. Match the theme to the article/topic mood.
